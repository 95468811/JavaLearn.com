甚麼是變數？

變數的定義是:用來命名一個數據的名子。你可以先暫時理解為幫資料取名子

EX:你把你的小孩叫做小明，那妳以後呼叫小明就是指你的小孩

來看個實例吧！

int year = 1949;

year 是標示符
= 是賦值操作符(不是數學上的=)
1949 是數字類型的值
; 代表該行結束

簡單理解為你創造一個名為year的數字"1949"。

變數的類型

一個變數的類型，決定它可以裝載哪種值。Java中一共有8種基本型態，大概可以拆分成

整數：+-數字


浮點數(小數)：float(小數點後一位)，double(小數點後2位以上)


布林值：存放判斷需要用的true(1)與false(0)


字元：單一字
字元(char)只能存放單一個字，值要用單引號''表示。例如'A'或'嗨'


字串：多個字連在一起
字串(String)並不是基本類型！但他的使用次數和基本類型不相上下，值要用雙引號""表示。例如"你好"或"hellow world"

命名規則

1.變數的命名只能用"數字、字母、$、_"等，且變數名的第一個字不能為數字。

2.盡量使用完整的單字命名，而不是縮寫。比如name、years，不要用縮寫n、y。

3.不能用關鍵字(int、class等等)，但是可以包含關鍵字。比如char class1 = 'A';
