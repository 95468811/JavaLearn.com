流程控制

寫java程式就像說英文一樣，但他有自己的規則。讓我們來看最基本的流程控制吧

if - else語法



int a = 10;

if(int a == 10) {      //如果a的值是10的話，就顯示a = 10
  System.out.println("a = 10");
} else if(int a > 10) {    //如果a的值大於10的話，就顯示a > 10
  System.out.println("a > 10");
} else {   //如果上述條件都不成立的話，就顯示都不符合
  System.out.println("都不符合");
}

switch - case語法

跟if-else很像，但每段case結束後都要執行break。否則會連下面的其他程式一起執行

int a = 10;

switch(a) {    //如果a的值是0的話就顯示0
case 0:
 System.out.println("0");
 break;
case 10:   //如果a的值是10的話就顯示10
 System.out.println("10");
 break;
default:   //可寫可不寫，用途跟else一樣
 System.out.println("都不是");
}

while迴圈

只要設定的條件為true的話，就會一直執行迴圈裡的程式；false就不執行。

int a = 1;

while(a <= 3) {  //只要a的值小於等於3就繼續執行
 System.out.println("執行了"+a+"次");
 a++;    //每次結束a的值+1
}

顯示結果：
執行了1次
執行了2次
執行了3次

do-while迴圈

和while迴圈一樣，只不過他會"先執行"在判斷true和false。

int a = 1;

do {
 System.out.println("執行了"+a+"次");
 a++;
} while(a < 1);   //a的值小於1才繼續執行

顯示結果：
執行了1次

for迴圈

從java2~6都很常用到的迴圈，非常便利。

for(int i = 0;i <= 3;i++) {   //建立迴圈的同時聲明變數i的值為0，只要i小於等於3就繼續執行，每次執行結束i的值+1
  System.out.println("執行了"+i+"次");
}

顯示結果：
執行了1次
執行了2次
執行了3次

對迴圈的控制

在迴圈內可以使用break、continue等關鍵字來控制迴圈

break;   //立即結束當前程式
continue;  //立即在執行一次當前迴圈。
