運算式

算數操作符

顧名思義就是+(加)-(減)*(成)/(除)以及++(自增)--(自減)。但++和--的優先級會更大

int a = 10;
int b = 5;
int c = a+b;
最後c的值就會是15

int a = 10;
int b = 5;
int a++;
int b--; 最後a的值會是11，b是4

關係操作符

不外乎就是比較變數之間的關係：

>大於
<小於
>=大於或等於
<=小於或等於
==是否相等
!=是否不相等

邏輯運算符

用來操控程式運算的

&&包含、還有
||或是
!取反值

int a = 10;
int b = 5;
if(a > b && a == 10) //一定要a大於b而且a等於10才會執行
if(a > b || a ==10) //只要a大於b或是a等於10就能執行
if(!(a > b || a ==10)) //如果a"不"大於b或是a"不"等於10就能執行

賦值操作符

=賦值操作
+=對自己運算，並賦值
-=同上
*=同上
/=同上
%=同上

int a = 10; //把10放進a裡面
int b = 5;
int a += b; //把a+b在放進a裡面，故a等於15
int a /= b; //把a/b放進a裡面，故a等於2
int a %= b; //把a/b的餘數放進a裡面，故a等於0
