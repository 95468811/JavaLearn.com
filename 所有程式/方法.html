方法的建立規則是這樣：

訪問修飾符 修飾符(可無) 回傳值 方法名(參數(可無)) {}

public staitc void add(int a) {;   //一個有參數，沒有回傳值的靜態方法
 System.out.println("add增加了"+a);
}

public int autoAdd() {;   //一個有回傳值，但沒參數的一般方法
 System.out.println("autoAdd增加了5");
 return 5;   //returm回傳了5給方法，故此方法的值就是5
}

public staitc void main(String[] args) {
 add(10);
 System.out.println(autoAdd());
}

add增加了10
autoAdd增加了5
5

方法的重寫(Overload)

Overload就是指一個方法除了參數外其他都一樣。

方法的覆寫(Override)

Override就是指一個方法除了內容外其他都和父類一樣。
