陣列(Array)

陣列是指有一個固定長度的容器，裡面裝著相同型態的值。比如100ml的水壺只能裝水，不能裝果汁

創建一個array

int[] a = new int[5];   //創建一個只能裝5個int的陣列
float[] b = new float[]{};  //創建一個可以無限裝float的陣列(浪費空間...)
char[] c = new char[3]{'認','真','學'};  //創建array的同時賦值

array的訪問與賦值

陣列是一格一格連續的空間，從0開始算起。也就是說長度是5的陣列，格子的名稱會是0~4！

int[] a = new int[5];
a[0] = 1;  //第0格放1
a[1] = 2;
a[2] = 3;
a[3] = 4;
a[4] = 5;
System.out.println(a[4]);  //顯示結果為5
System.out.println(a.length);  //顯示陣列a的長度，故為5
