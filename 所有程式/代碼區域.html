代碼區域

代碼區域

程式碼都有屬於自己的作用域，到了作用域外就會被消除。

修飾符

修飾符可以在聲明變數或方法做使用，他能為你的程式增加一點規則。

static int a = 10;=   //(靜態)所有程式都共用這個a，可以跳脫一些作用域的限制
final int a = 10;   //(常數)a的值在聲明時就固定了，其他程式無法再更改
public int a = 10;   //(公有的)外部的任何程式都可以直接存取a
private int a = 10;   //(私有的)a只給作用域李的程式存取
protected int n = 10;   //(受保護的)a只給同個資料夾做存取
package java3;   //(包)讓電腦知道目前的檔案在java3的資料夾裡(通常不寫)
